# rename.py: 데이터 폴더 및 파일 이름 변경 스크립트 사용법

1. 데이터 폴더 이름 변경
   - 각 날짜 폴더 아래에 있는 클래스 폴더 이름을 다음과 같이 변경합니다:
     - dw -> 0
     - 10^1 -> 1

2. 날짜 폴더 설정 및 스크립트 실행
   - 스크립트에서 base_dir 변수를 데이터 폴더 경로로 설정합니다:
     base_dir = "/home/jijang/projects/Bacteria/dataset/B_cereus/240311"

3. 스크립트를 실행하여 비디오 및 yuv 파일들의 이름을 변경합니다.

4. 접두사 매핑
   - 폴더명에 접두사를 추가하여 구분합니다:
     0 -> dw_
     1 -> e102_
     2 -> e103_
     3 -> e104_
     4 -> e105_
